StatusDeLocalizacao

StatusDeLocalizacao class Null safety

StatusDeLocalizaçao gerencia os status da localização no aplicativo. É necessário que todas requests relacionados à localização venham daqui pois não se pode ter requests de localização simultaneas.

Automaticamente procura saber o status atual do serviço de localização a cada X segundos, onde X é determinado pelo intervaloEntreUpdates. Se intervaloEntreUpdates for 0, o timer nunca roda.

! AVISO, ESSA CLASSE É PRA SER UM SINGLETON, é só que usar o pattern aqui... eu achei meio complicado de se fazer sem tirar a opção de mudar o tempo do timer no constructor.
